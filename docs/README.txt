Publication link workflow: each publication entry has a link box below it. Your RA
can progressively fill in paperUrl, pdfUrl, anthologyUrl, arxivId, doi, websiteUrl,
codeUrl, datasetUrl, videoUrl, and slidesUrl fields in src/data/publications.json.
Entries without any links show "No links yet" so you can see at a glance what still
needs attention.
